# An Application for Mental Arithmetic

Консольное приложение на Python для тренировки устного счёта.

## 📌 Описание

Данное приложение предназначено для практики устной арифметики.  
Программа генерирует математические задачи, которые пользователь решает в интерактивном режиме.

Подобные приложения используются для развития скорости мышления и вычислительных навыков :contentReference[oaicite:0]{index=0}.

---

## ✨ Возможности

- Генерация арифметических задач
- Ввод ответа пользователем
- Проверка правильности ответа
- Работа с циклами (`while`, `for`)
- Обработка ошибок (`try/except`)
- Простая логика взаимодействия с пользователем

---

- Python
